# ops

Running and debugging xmake itself — environment variables, themes, troubleshooting, and developing xmake's own source tree.

| Skill | Topic |
| --- | --- |
| [xmake-env-vars](./xmake-env-vars/) | `XMAKE_*` environment variables — paths, behavior, `XMAKE_RCFILES` |
| [xmake-theme](./xmake-theme/) | `xmake g --theme=...` — fixing garbled output, choosing a theme |
| [xmake-troubleshooting](./xmake-troubleshooting/) | Hangs (`XMAKE_PROFILE=stuck`), profiling, `cli.bisect`, `-vD` |
| [xmake-dev](./xmake-dev/) | Build xmake from source, `srcenv.profile`, debug Lua + C core |
