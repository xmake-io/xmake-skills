# packages

Everything about C/C++ packages — standalone `xrepo`, virtual environments, writing/testing recipes, private repos, and download acceleration.

| Skill | Topic |
| --- | --- |
| [xrepo-cli](./xrepo-cli/) | Standalone package manager: install/search/info/fetch/export |
| [xrepo-env](./xrepo-env/) | `xrepo env` virtual environments (ad-hoc, project-local, named) |
| [xmake-network](./xmake-network/) | Proxy / `pac.lua` / mirror config for fast downloads, esp. in China |
| [xmake-repo-testing](./xmake-repo-testing/) | Test package recipes in xmake-repo via `scripts/test.lua` |
| [xmake-debug-package-source](./xmake-debug-package-source/) | Debug broken packages — `-d <dir>`, iterate on source, extract patches |
| [xmake-private-packages](./xmake-private-packages/) | Local / remote / in-project private package repositories |
