# testing

Writing and running tests — both for *your* C/C++ project and for xmake's own test suite when hacking on xmake itself.

| Skill | Topic |
| --- | --- |
| [xmake-tests](./xmake-tests/) | `add_tests` in your project, `xmake test`, gtest/catch2/doctest integration |
| [xmake-unit-tests](./xmake-unit-tests/) | Xmake's own test suite: `xmake l tests/run.lua [name]`, writing new tests |
